Source column Target column Method Weight Tolerance % Group
Word match and Similarity already ignore case, punctuation, and common legal-form suffixes (GmbH, AG, Inc., Ltd., ...), so "ACME Trading GmbH" and "acme trading" compare as equal. Tolerance % only applies to the Numeric method.
Group: give two or more fields the exact same group name to combine them into a single composite key — e.g. street + postal code, or manufacturer + part number. A strong match on one member of a group can never make up for a mismatch on another: the same street name in a different city, or the same part number from a different manufacturer, correctly scores as no match. Leave empty for fields that should count on their own.
