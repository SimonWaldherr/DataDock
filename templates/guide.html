Import Notes

FormatBehavior
CSV / TSVHeader row can be auto-detected, forced present, or forced absent.
JSON / NDJSON / MessagePack / CBOR / BSONImports objects into columns derived from object keys.
HTMLImports table rows from HTML documents into a rectangular table.
SQLiteImports rows from user tables/views into one combined table with a source_table column.
Parquet / Arrow / Feather / DuckDBImports file-level manifests so the files can be tracked in DataDock.
iCalendar / vCardImports events/tasks or contacts into queryable columns.
GeoJSON / GeoPackage / GPX / KML / OSMImports features with GeoJSON geometry text, geometry type, and useful tags/properties.
OSM PBFImports tagged nodes and ways from compact binary OSM extracts, building LineString/Polygon geometry from node references where possible.
Shapefile ZIPReads the first .shp in a ZIP, converting shapes to GeoJSON geometry and DBF attributes to columns.
MBTilesImports metadata plus a tile index with size, SHA-256, and base64 tile data for small tiles.
RGImports JSON/NDJSON routing graphs as node and edge rows with geometry where present.
XLSXReads the first worksheet and imports it through the CSV path.

Useful Starter Queries

SELECT * FROM datadock_demo_people;
SELECT d.name AS department, COUNT(p.id) AS people
FROM datadock_demo_departments d
JOIN datadock_demo_people p ON p.department_id = d.id
GROUP BY d.name;
SELECT p.name, pr.name AS project, pr.status, pr.budget
FROM datadock_demo_people p
JOIN datadock_demo_projects pr ON pr.owner_id = p.id;
SELECT project_id, SUM(amount) AS event_total
FROM datadock_demo_events
GROUP BY project_id;
SELECT pr.name, SUM(i.amount) AS invoiced
FROM datadock_demo_projects pr
JOIN datadock_demo_invoices i ON i.project_id = pr.id
GROUP BY pr.name;
SELECT name, category, lon, lat
FROM datadock_demo_locations
ORDER BY name;
SELECT source, external_id, imported_at, payload_json, payload_xml
FROM datadock_demo_payloads
ORDER BY imported_at;
PRAGMA table_info(datadock_demo_people);
